# Backend

The backend implements the necessary logic to create encrypted records, decrypt encrypted records and create/verify zk-SNARKs.

## API

### `GET /api/record/create`

Create a new record

### `GET /api/record/invalidate`

Invalidate a record

### `GET /api/zk/create`

Create a zk-SNARK

### `GET /api/zk/verify`

Verify a zk-SNARK
